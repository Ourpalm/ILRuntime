贡献指南

本文档描述了如何对ILRuntime提供的贡献情况。

Pull Request

  • 不要进行较大内容的PR.
  • 每个PR只做一件事情
  • 确保PR的代码能够通过编译
  • 提交PR时，请务必保证针对代码的所有测试都必须通过
  • 提交PR时，请务必记录好PR的原因
